# 🐾 CursorPet (마우스 커서 펫)

> 내 마우스 커서를 졸졸 따라다니는 귀여운 데스크톱 펫 프로그램입니다! 

![Python Version](https://img.shields.io/badge/python-3.8%2B-blue)
![PyQt6](https://img.shields.io/badge/UI-PyQt6-brightgreen)

CursorPet은 윈도우 화면 위에서 마우스 커서의 움직임을 감지하여 캐릭터가 최상단에서 자연스럽게 따라다니도록 구현된 PyQt6 기반의 오픈소스 데스크톱 토이입니다. 우클릭 메뉴를 통해 크기와 간격, 그리고 캐릭터 스킨을 실시간으로 변경할 수 있습니다.

---

## ✨ 주요 기능
- **마우스 추적:** 마우스 좌표를 실시간으로 가져와 펫이 자연스럽게 따라옵니다.
- **실시간 캐릭터 변경:** `characters` 폴더 안에 원하는 `.png` 이나 `.gif` 이미지를 넣고 우클릭 메뉴로 즉시 교체할 수 있습니다.
- **다양한 커스터마이징:** 펫의 크기 배율과 마우스와의 거리를 조절할 수 있습니다.
- **시스템 트레이 아이콘:** 화면을 가리지 않도록 윈도우 트레이 영역에서 우클릭 메뉴를 제공합니다.
